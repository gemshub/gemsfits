GEMSFIT2

About GEMSFIT2 code
  • Data Source - Gemsfi2 uses local stored EJDB database generated using csvtoejdb tool
  • Data Select - The user can select individual experiments, and/or exclude datasets, and temperature and pressure intervals by editing the input file acordingly.
  • The residuals are calulated based on user selected functions. Severeal weighting scheems are also available.
  • Data To Compare - phase chemical (elements) composition, given in molal or loga (meaning log) units; phase dependent components amount/fraction; phase properties: pH, Eh, amount, volume, RHO, alkalinity, surface area;

  • Parameters to fit - <G0> (Gibbs free energy at 25 C and 1 bar), <bIC> (bulk composition), <PMc> (interaction parameter coefficients for non-ideal solutions), <DMc>, <fDQF>, <bIC>. The G0 parameters can be also reaction constrined and the bIC ones can be linked to each other (eg. titration).

  • The optimization algorithms are provided in the NLopt library.

  • The statistics represent the goodness of fit (Coefficient of determination R^2, Pearson's Chi Square test, Reduced_chi_square) but fit independent statistics with sensitivity analysis, Composite Scaled Sensitivities, Variance-Covariance matrix, Parameter Standard Deviations/Errors, Confidence intervals, Coefficient of Variation, Parameter t-statistic, Correlation matrix. For details: Mary C. Hill, Claire R. Tiedeman (2007) Effective Groundwater Model Calibration: With Analysis of Data, Sensitivities, Predictions, and Uncertainty. 480 pages.
