2. Sensitivity analysis and paramater stitistics

A. Composite Scaled Sensitivities:
        See ref. [1] section 4.3.4. If there is no weighting the weight is asumed 1 for all measured values. Larger values indicate parameters for which observations provide more information. A value less than 1 or less than         (or close to) 1% of the highest value, means that the parameter is poorly estimated. These parameters should be fixed during regression.
B. Variance-Covariance matrix:
 See ref. [1] section 7.2. V(b) = s^2([X]trans*[X])^1;  [W] - weight matrix is considered as 1 as there is no full weight matrix calculted, based on true experimental error (not provided for all experiments). X - sensitivity matrix. When the chosen weight is other than inverr (1/sigma), inverr3 (1/measured^2) or inverr_norm (w=1/(error/<meas(average)>)^2), the error variance (s^2) is assumed 1. For inverr2, the error variance is calculated as weighted_Tfun_sum_of_residuals/degrees of freedom.
