GEMSFITS Code for GEM Input Parameter Optimization

Technical information

Contents

  • external libraries: have to be installed before building the code
  • build in ubuntu 14.04
  • standalone libraries: come together with gemsfit2 code in predefined directory structure

GEMSFIT2 external libraries:

  • boost - search for libboost and istall libboost-all-dev. Ot in terminal: sudo apt-get install libboost-all-dev

  • search for liblapack and install liblapack3gf and liblapack-dev.

  • check if libblas and install are installed by liblapack installer, if needed, install libblas3gf and libblas-dev.

  • omp - should already be available as omp.h in C/C++ http://en.wikipedia.org/wiki/OpenMP

  • armadillo - search for armadillo in the package manager and install armadillo-dev and libarmadillo2 

  • Nlopt - Go to http://ab-initio.mit.edu/wiki/index.php/NLopt#Download_and_installation and download nlopt-2.2.4.tar.gz

                Unpack it, go into the nlopt directory and execute
                ./configure
                make
                sudo make install

  • jansson - http://www.digip.org/jansson/

                Unpack it and execute

                ./configure
                make
                make check
                make install

Build GEMSFITS in ubuntu 14.04 64bit LTS:

Execute the following commands to install the necessary libraries:

~$ sudo apt-add-repository ppa:comp-phys/stable
~$ sudo apt-get update
~$ sudo apt-get install g++ libboost-all-dev liblapack3gf liblapack-dev libarmadillo-dev libnlopt0 libnlopt-dev libjansson4 libjansson-doc libjansson-dev libjansson-dbg

For Qt5 execute the following command as well:

~$ sudo apt-get install libgl1-mesa-glx libgl1-mesa-dev

GEMSFIT2 standalone libraries

  • GEMS3K

  • csvtoejdb

  • EJDB - http://ejdb.org/

  • muparser - http://muparser.beltoforion.de/
