1. Results window

Save gemsfit2 results to record and Update record with saved results .  By saving the results to the database, they can be latter accessed in the Fit results view mode .

  • In the View Results window, the gemsfit2 output is displayed in several tabs in a spreadsheet format. The output contains:

  • Fitted Parameters tab with the initial and fitted values of the parameters and the associated parameter statistics;

  • Fit for Samples tab containing the dependent/calculated and measured properties, residuals, as well as the weights and other user pre-defined output using ADDOUT;

  • Sum. Statistics tab with the summary statistics describing the goodness of fit;

  • Sensitivity Data tab containing the calculated sensitivities for each measured value per each parameter;

  • Quantile Residual tab with the ordered residuals and their respective quantiles (points taken at regular intervals form the cumulative distribution function of the residuals);

  • MC results tab containing the resulted parameter values for each Monte Carlo optimization run;

  • Inverse Modeling Results tab showing the result of an executed inverse modeling problem and the and results for the nested function parameters;

  • Tew log tab containing some diagnostic information related to the fitting run.

2. Plotting the results

The values  from any results  window tab can be plotted in XY plots. To do this one can select the desired X-axis and multiple Y-axis using the right mouse click and chose Toggle X or Toggle Y on the desired tab columns. If this columns were correctly set  (x) or (y) will appear at the end of the column name. To plot the selected column values click Plot parameter optimization results . The chosen plot settings are saved together with the optimization results in the database (collection "fits") upon clicking Update record with saved results .
